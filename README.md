# Machine Learning Classification Study (KNN, SVM & Decision Tree)

## Overview

This repository contains my personal study notebook focused on three fundamental Machine Learning classification algorithms:

- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree

The notebook was created as part of my Machine Learning learning journey.

Its primary purpose is to strengthen my understanding of classical supervised learning algorithms through hands-on practice, experimentation, visualization, and detailed notes.

## Installation

Clone the repository

```bash
git clone https://github.com/brainExplorer/knn-svm-decision-tree-study.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
knn_svm_dt_df.ipynb
```
